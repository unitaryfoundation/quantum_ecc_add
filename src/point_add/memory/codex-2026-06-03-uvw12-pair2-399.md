## Caveat

`KAL_UVW_TIGHTEN=14` default. The full harness is clean for the Fiat-Shamir test
set, but adjacent settings still fail non-monotonically by phase or classical
mismatch. Do not assume nearby values are safe without re-running all 9024 shots.
